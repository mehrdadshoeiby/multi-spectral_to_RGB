Enviorment: python3 Anocoda2 Pytorch 0.4.0

The test and valid code are provided in this folder.
The MRAE of valid dataset is 0.1195

The input data are all converted to .mat version and provided in this folder.

You can run the valid.sh to generate the reconstructed hyperspectral images in valid dataset.
You can run the test.sh to generate the reconstructed hyperspectral images in final test dataset.

The output data are all in .mat version.
You can use the convert.m to get the final .fla data.
